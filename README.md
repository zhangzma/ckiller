Ckiller 是什么？
=================
  本项目主要是关于《C++ Primer 4th》该书的练习。
  他应该不会是别的什么东西。

# 编译
## 单文档程序
   大多数的程序都是单源文件程序，它不会太复杂。所采用的编译为是 *gcc 4.5.3*。
   编译命令如下：
>    g++ prog.cc -o prog

## 多文档程序
   会提供相应的 *makefile*。
   
# 编码风格
  1. 本项目编码风格参照《C++ Primer》进行；
  2. 文件命名采用Linux风格；
  	 练习命名文件举例：*ex7-13.cc*
